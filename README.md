# Sistema de Cadastro de Usuários

Projeto desenvolvido em Python com integração MySQL.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Conexão com banco de dados MySQL
- Manipulação de tabelas

## Tecnologias utilizadas
- Python
- MySQL
- VS Code

## Objetivo
Projeto criado para praticar desenvolvimento back-end e banco de dados.

## Como executar
1. Instale o Python
2. Instale as dependências:
pip install mysql-connector-python

3. Execute o arquivo:
python main.py
